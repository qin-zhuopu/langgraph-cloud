# 自动询比价智能体 — 采购员信息维护需求文档

**作者：** 15481
**日期：** 2026-04-15
**模块：** 自动询比价智能体 > 采购员信息维护

---

## 概述

采购员信息维护为采购员提供个人工作配置的统一管理页面。页面采用卡片式布局，分为五个配置区域：基本信息（只读）、邮箱配置、邮件服务器配置、供应商推荐个数配置、ERP请购池数据抓取设置。所有可编辑配置通过页面底部统一的【保存所有设置】按钮一次性提交，同时提供【测试邮箱连接】按钮验证邮箱配置的有效性。

---

## 需求1：基本信息展示

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 采购员工号 | 文本框（只读） | - | 自动显示当前登录采购员的工号，不可编辑 |
| 采购员姓名 | 文本框（只读） | - | 自动显示当前登录采购员姓名，不可编辑 |
| 工作邮箱 | 文本（展示） | - | 显示已配置的工作邮箱地址 |
| 邮箱配置状态 | 标签（展示） | - | 显示"已配置"（绿色标签）或"未配置"（黄色标签） |
| 最近更新时间 | 日期时间（展示） | - | 显示最近一次修改配置的时间，格式为yyyy-mm-dd HH:mm:ss |

### 逻辑说明

1. 以卡片形式展示当前采购员的基本信息，仅显示当前登录采购员自己的数据。
2. 采购员工号和采购员姓名为只读输入框样式，数据来源于系统登录信息，不可编辑。
3. 工作邮箱显示当前已配置的邮箱地址，邮箱授权码不在此区域明文展示。
4. 邮箱配置状态以标签形式展示：已配置（绿色）、未配置（黄色）。

---

## 需求2：邮箱配置

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 工作邮箱 | 文本框 | 必填 | 手动输入邮箱地址，需校验邮箱格式合法性 |
| 邮箱授权码 | 密码框 | 必填 | 手动输入邮箱授权码，输入时以密文显示 |

### 逻辑说明

1. 以卡片形式展示邮箱配置区域，采购员可直接在页面上编辑工作邮箱和邮箱授权码。
2. 邮箱地址需校验格式合法性，格式不正确时提示"邮箱格式不正确"。
3. 邮箱授权码在输入时以密文显示，提交后系统加密存储，不得明文保存。
4. 必填字段未填写时，阻止保存并提示用户填写。

## 需求3：邮件服务器配置

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| SMTP服务器地址 | 文本框 | 必填 | 手动输入，如smtp.example.com |
| SMTP端口 | 数字框 | 必填 | 手动输入，常用端口：25/465/587 |
| SMTP加密方式 | 下拉单选 | 必填 | 下拉选项：无/SSL/TLS |
| IMAP服务器地址 | 文本框 | 必填 | 手动输入，如imap.example.com |
| IMAP端口 | 数字框 | 必填 | 手动输入，常用端口：143/993 |
| IMAP加密方式 | 下拉单选 | 必填 | 下拉选项：无/SSL/TLS |

### 逻辑说明

1. 以卡片形式展示邮件服务器配置区域，采用两列网格布局。
2. 系统可预置常见邮箱服务商的服务器配置（如QQ邮箱、163邮箱、企业微信邮箱等），采购员选择邮箱类型后自动填入服务器信息。
3. 采购员也可手动输入自定义的邮件服务器信息。
4. 必填字段未填写时，阻止保存并提示"请填写所有邮件服务器配置"。

## 需求4：供应商推荐个数配置

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| ERP内部推荐个数 | 数字框 | 必填 | 设置ERP内部供应商推荐展示的最大个数，默认5，范围0~10的整数 |
| 外部联网推荐个数 | 数字框 | 必填 | 设置通过搜索引擎和天眼查等平台推荐展示的最大个数，默认5，范围0~10的整数 |

### 逻辑说明

1. 以卡片形式展示供应商推荐个数配置区域。
2. ERP内部推荐个数：控制从Oracle EBS其他业务实体中推荐的供应商最大展示数量。
3. 外部联网推荐个数：控制通过搜索引擎和天眼查等第三方平台推荐的供应商最大展示数量。
4. 两个字段均为必填，默认值为5。
5. 输入值必须为0到10的整数，超出范围时提示"推荐个数需在0~10之间"。
6. 字段后方显示"个（0~10）"提示文字。

## 需求5：ERP请购池数据抓取设置

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 询价单生成方式 | 单选按钮 | 必填 | 选项：自动生成询价单 / 先展示明细再生成询价单。默认"先展示明细再生成询价单" |
| 报价单超期月份 | 数字框 | 必填 | 设置报价单创建日期超过多少个月后对应物料被抓取进行询价，默认6，范围0~12的整数，单位：月 |

### 逻辑说明

1. 以卡片形式展示ERP请购池数据抓取设置区域。
2. **询价单生成方式：**
   - 选择"自动生成询价单"：点击【自动创建询价单】后，系统直接按业务实体+四级品类分组生成询价单，不展示预览列表。
   - 选择"先展示明细再生成询价单"（默认）：点击【自动创建询价单】后，系统先展示从ERP抓取的物料数据预览列表，采购员勾选确认后再生成询价单。
3. **报价单超期月份：** 系统抓取ERP请购池数据时，仅抓取请购池物料对应报价单创建日期超过设定月份的物料数据。例如设置为6，则仅抓取报价单创建超过6个月的物料。
4. 输入值必须为0到12的整数，超出范围时提示"报价单超期月份需在0~12之间"。
5. 默认值为6个月，字段后方显示"月（0~12）"提示文字。

---

## 需求6：统一保存与邮箱连接测试

### 逻辑说明

1. 页面底部设置统一操作栏，包含【保存所有设置】和【测试邮箱连接】两个按钮，操作栏位于所有配置卡片下方，以分隔线与配置区域分隔。
2. **保存所有设置：** 点击后对所有可编辑字段进行统一校验，校验通过后一次性保存所有配置项。校验规则包括：
   - 工作邮箱不为空且格式合法
   - 邮箱授权码不为空
   - 邮件服务器所有必填字段不为空
   - ERP内部推荐个数和外部联网推荐个数在0~10范围内
   - 报价单超期月份在0~12范围内
3. 校验不通过时，提示具体的错误信息，阻止保存。
4. 保存成功后提示"所有设置保存成功"，同时更新基本信息区域的邮箱配置状态和最近更新时间。
5. **测试邮箱连接：** 点击后使用当前填写的邮箱地址和授权码尝试连接邮件服务器。
   - 若邮箱或授权码未填写，提示"请先填写邮箱和授权码"。
   - 测试SMTP发送连接和IMAP接收连接。
   - 连接成功提示"邮箱连接测试成功"。
   - 连接失败提示具体错误原因（如授权码错误、服务器不可达等）。
