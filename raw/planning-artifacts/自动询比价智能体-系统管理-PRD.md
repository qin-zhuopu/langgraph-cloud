# 自动询比价智能体 — 系统管理需求文档

**作者：** 15481
**日期：** 2026-04-15
**模块：** 自动询比价智能体 > 系统管理

---

## 概述

系统管理模块供IT管理员使用，用于管理用户账号、角色权限分配、系统参数配置及询价单模版维护。系统采用RBAC权限模型。

---

## 需求1：用户账号管理

### 用户列表字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 工号 | 文本（展示） | - | 显示用户工号 |
| 姓名 | 文本（展示） | - | 显示用户真实姓名 |
| 角色 | 文本（展示） | - | 显示用户角色：采购员/IT管理员 |
| 账号状态 | 开关（展示/操作） | - | 开关形式展示启用/禁用状态，IT管理员可直接切换 |
| 创建时间 | 日期时间（展示） | - | 格式为yyyy-mm-dd HH:mm |

### 逻辑说明

1. 以列表形式展示所有用户账号信息。
2. 列表支持分页展示。
3. 支持按姓名、角色进行筛选。

---

## 需求2：用户账号创建

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 工号 | 文本框 | 必填 | 手动输入，全局唯一 |
| 姓名 | 文本框 | 必填 | 手动输入用户真实姓名 |
| 初始密码 | 密码框 | 必填 | 手动输入，以密文显示 |
| 角色 | 下拉单选 | 必填 | 下拉选项：采购员、IT管理员 |

### 逻辑说明

1. 点击【新建用户】按钮，弹出创建表单。
2. 填写用户基本信息并分配角色。
3. 工号全局唯一，重复时提示"该工号已存在"。
4. 必填字段未填写时，阻止提交并提示用户填写。
5. 创建成功后用户可使用工号和初始密码登录系统。

---

## 需求3：用户账号编辑

### 逻辑说明

1. 点击【编辑】按钮，弹出编辑表单，回显当前用户数据。
2. 工号不可修改。
3. 姓名、角色允许修改。
4. 支持重置用户密码。
5. 必填字段未填写时，阻止提交并提示用户填写。

---

## 需求4：用户账号启用/禁用

### 逻辑说明

1. 用户列表中账号状态以开关形式展示，开启为"启用"，关闭为"禁用"。
2. IT管理员点击开关切换状态时，弹窗提示"是否确认启用/禁用该用户？"。
3. 点击【确定】，该用户账号状态变更，禁用后用户无法登录系统。
4. 点击【取消】，开关恢复原状态，不做操作。
5. 禁用IT管理员角色账号时，系统需检查是否还有其他启用状态的IT管理员。若禁用后将无启用状态的IT管理员，则阻止操作并提示"系统中至少需要保留一个启用状态的IT管理员"。

---

## 需求5：系统参数配置

### 参数说明

| 参数名称 | 参数类型 | 默认值 | 参数说明 |
|---------|---------|--------|---------|
| 报价抓取周期 | 数字（分钟） | 30 | 定时抓取供应商报价邮件的执行间隔 |
| 询价单编号前缀 | 文本 | RFQ | 询价单编号的前缀字符 |
| 邮件发送重试次数 | 数字 | 3 | 邮件发送失败后的最大重试次数 |
| ERP接口超时时间 | 数字（秒） | 30 | Oracle EBS API调用的超时时间 |
| ERP接口重试次数 | 数字 | 3 | Oracle EBS API调用失败后的最大重试次数 |
| 附件大小限制 | 数字（MB） | 20 | 单个附件文件的最大大小 |

### 逻辑说明

1. IT管理员可查看和修改系统参数配置。
2. 修改参数后立即生效，无需重启系统。
3. 参数值需校验合法性（如数字类型不允许输入文本）。
4. 提供【恢复默认值】按钮，可将所有参数恢复为默认值。

---

## 需求6：用户搜索与筛选

### 逻辑说明

1. 用户列表上方提供搜索和筛选区域。
2. 支持按工号、姓名模糊搜索。
3. 支持按角色、账号状态进行筛选。
4. 筛选条件支持组合使用。
5. 点击【重置】按钮清空所有筛选条件。

---

## 需求7：询价单模版维护

### 模版列表字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 模版名称 | 文本（展示） | - | 显示模版名称 |
| 四级品类 | 文本（展示） | - | 显示模版关联的四级品类 |
| 模版描述 | 文本（展示） | - | 显示模版用途说明 |
| 模版文件 | 文件（展示） | - | 显示已上传的模版文件名 |
| 状态 | 开关（展示/操作） | - | 开关形式展示启用/禁用状态，IT管理员可直接切换 |
| 创建人 | 文本（展示） | - | 显示创建该模版的操作人 |
| 创建时间 | 日期时间（展示） | - | 格式为yyyy-mm-dd HH:mm |
| 更新时间 | 日期时间（展示） | - | 显示最近一次更新时间 |

### 模版创建/编辑字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 模版名称 | 文本框 | 必填 | 手动输入模版名称，全局唯一 |
| 四级品类 | 下拉选择器 | 必填 | 选择模版关联的四级品类 |
| 模版描述 | 文本框 | 非必填 | 手动输入模版用途说明 |
| 模版文件 | 文件上传 | 必填 | 上传询价单模版文件（支持Excel、Word格式），模版中可包含占位符用于自动填充询价单数据 |

### 逻辑说明

1. 以列表形式展示所有询价单模版。
2. IT管理员可新建模版：填写模版名称、选择四级品类、填写描述，上传模版文件。
3. 模版文件支持Excel和Word格式，模版中可使用占位符（如{询价单号}、{物料明细}、{供应商信息}等），发送询价时系统自动将占位符替换为实际数据生成询价单文件。
4. IT管理员可编辑已有模版的名称、四级品类、描述，可重新上传模版文件。
5. IT管理员可启用或禁用模版，禁用的模版不在采购员的询价单模版下拉框中显示。
6. 模版名称全局唯一，重复时提示"模版名称已存在"。
7. 必填字段未填写时，阻止提交并提示用户填写。
8. 删除模版前弹窗确认，已被询价单引用的模版不允许删除，仅允许禁用。
