# 自动询比价智能体 — 询价单详情需求文档

**作者：** 15481
**日期：** 2026-04-15
**模块：** 自动询比价智能体 > 询价单详情（含AI智能供应商推荐、AI比价与策略推荐）

---

## 概述

询价单详情是采购员对询价单进行全流程操作的核心页面。采购员可在此维护询价单信息、发送询价邮件、查看供应商报价、进行多轮询价、生成报价对比表，并最终将结果导入ERP。同时集成AI智能供应商推荐和AI比价与策略推荐功能。

页面采用折叠面板（accordion）布局，分为物料明细、询价单附件、供应商明细、报价对比表、邮件记录五个可折叠区域。抬头信息卡片包含保存按钮，采购员可随时保存修改。

询价单为头行结构，抬头为询价单基本信息，行分为物料明细行和供应商明细行。

## 需求1：询价单抬头信息维护

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 报价单号 | 文本（只读） | - | 系统自动生成，不可编辑 |
| 询价单标题 | 文本框 | 必填 | 手动输入或系统根据物料品类自动生成，允许编辑 |
| 采购员 | 文本（只读） | - | 自动显示当前登录采购员，不可编辑 |
| 询价单状态 | 文本（只读） | - | 系统自动维护，详见状态流转说明 |
| 报价截止时间 | 日期时间选择器 | 必填 | 格式为yyyy-mm-dd HH:mm:ss，允许编辑，必须晚于当前时间 |
| 业务实体 | 文本（只读） | - | 来源于Oracle EBS，自动填入，不可编辑 |
| 当前轮次 | 数字（只读） | - | 系统自动维护，显示当前询价轮次 |
| 备注 | 文本框 | 非必填 | 手动输入，允许编辑 |
| 是否年度谈判 | 开关 | 非必填 | 默认否，允许编辑 |
| 询价单模版 | 下拉选择框 | 必填 | 创建时根据物料四级品类自动关联对应模版，采购员可手动修改选择其他已启用模版 |
| 创建人 | 文本（只读） | - | 系统自动记录创建操作人 |
| 创建日期 | 日期（只读） | - | 系统自动记录，格式为yyyy-mm-dd |

### 逻辑说明

1. 报价单号由系统自动生成，全局唯一。
2. 采购员字段自动显示当前登录用户，不可修改。
3. 业务实体来源于Oracle EBS数据，自动填入。
4. 报价截止时间为采购员可编辑字段，精确到时分秒，用于告知供应商报价截止时间。
5. 询价单状态由系统根据操作自动流转。
6. 标记为"年度谈判"的询价单在列表中可通过筛选快速定位。
7. 询价单模版创建时由系统根据物料四级品类自动关联对应的已启用模版（模版维护中按四级品类配置），采购员也可手动修改选择其他模版。发送询价邮件时系统根据所选模版样式自动生成询价单文件作为邮件附件发送。
8. 发送询价或发起新一轮询价时，系统校验报价截止时间必须晚于当前时间，否则阻止发送并提示"报价截止时间必须晚于当前时间"。
9. 抬头信息卡片提供【保存】按钮，采购员修改抬头信息后点击保存，系统校验必填字段后保存成功。保存范围包括抬头信息、物料明细、供应商明细等所有可编辑内容。保存成功后提示"保存成功"。
10. 抬头操作按钮包括：保存、发送询价、截止报价、报价单导入ERP、中标结果导入ERP、作废、删除。

## 需求2：物料明细行维护

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 物料编码 | 文本框 | 必填 | 自动创建时由ERP带入；手工创建时手动输入 |
| 物料描述 | 文本框 | 必填 | 自动创建时由ERP带入；手工创建时手动输入或EBS带出 |
| 物料详细描述 | 文本框 | 非必填 | 允许编辑 |
| 数量 | 数字框 | 必填 | 必须大于0，允许编辑 |
| 单位 | 文本框 | 必填 | 自动创建时由ERP带入；允许编辑 |
| 需要日期 | 日期选择器 | 必填 | 格式为yyyy-mm-dd，允许编辑 |
| 四级品类 | 文本（只读） | - | 由系统根据物料编码从EBS自动获取，不可编辑 |

### 逻辑说明

1. 物料明细行以折叠面板内列表形式展示，支持新增和删除操作（无编辑按钮，字段直接在行内编辑）。
2. 自动创建的询价单，物料明细由ERP数据自动填入。
3. 手工创建的询价单，采购员手动录入物料信息。
4. 手工新增物料行时，采购员输入物料编码后，系统调用ERP接口自动获取物料描述、物料详细描述、单位、四级品类，这些字段不可手工编辑。接口调用参数：业务实体、物料编码。返参：物料描述、物料详细描述、单位、四级品类。
5. 四级品类字段由系统自动从Oracle EBS获取，不可手动编辑。
6. 数量字段为文本输入框（不显示上下箭头），仅允许输入数字和小数点。
7. 必填字段未填写时，阻止保存并提示用户。
8. 询价单发送后（状态为"询价中"及之后），物料明细行不允许新增、编辑和删除操作，仅可查看。

## 需求3：供应商明细行维护

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 供应商编码 | 文本框 | 必填 | 从供应商组自动匹配或手动输入 |
| 供应商名称 | 文本框 | 必填 | 从供应商组自动匹配或手动输入 |
| 供应商联系人 | 文本框 | 必填 | 从供应商组自动匹配或手动输入 |
| 联系人电话 | 文本框 | 非必填 | 从供应商组自动匹配或手动输入 |
| 供应商邮箱 | 文本框 | 必填 | 从供应商组自动匹配或手动输入，用于发送询价邮件 |
| 来源 | 文本（只读） | - | 显示供应商来源：供应商组匹配/手动添加/AI推荐 |

### 逻辑说明

1. 供应商明细行以折叠面板内列表形式展示，支持新增和删除操作（无编辑按钮，联系人、电话、邮箱字段直接在行内可编辑）。
2. 创建询价单时，系统根据物料四级品类自动匹配供应商组中的供应商，自动填入供应商明细行。
3. 采购员可以手工添加ERP系统中的供应商到询价单。
4. 采购员可通过AI智能供应商推荐功能添加供应商（详见需求9）。点击【智能推荐供应商】按钮弹出聊天式AI对话框，系统根据询价单物料的四级品类自动推荐ERP内部其他业务实体供应商和联网搜索供应商，采购员勾选后点击确定将选中供应商同步到供应商明细中。
5. 手工添加供应商时，采购员点击【手动添加供应商】按钮，系统在供应商明细列表末尾新增一行。采购员输入供应商编码后，系统调用ERP接口自动获取供应商名称、联系人、电话、邮箱。供应商名称不可手工编辑，联系人、电话、邮箱可在行内直接修改。来源标记为"手工添加"，报价状态为"未报价"。若供应商编码在ERP中不存在，提示"该供应商编码在ERP中不存在"。
6. 供应商邮箱为必填字段，用于发送询价邮件。
7. 来源字段自动标记供应商的添加方式。
8. 询价单发送后（状态为"询价中"及之后），供应商明细行不允许新增、编辑和删除操作，仅可查看。

## 需求4：询价单附件管理

### 逻辑说明

1. 采购员可上传与询价单相关的附件文件（技术规格书、图纸等）。
2. 支持常见附件格式：PDF、Excel、Word、图片等。
3. 上传的附件在发送询价邮件时作为邮件附件一并发送给供应商。
4. 采购员可查看、下载和删除已上传的附件。
5. 采购员点击【上传附件】按钮，弹出文件选择框，选择文件后上传。支持单个或多个文件上传，单个附件大小不超过系统参数配置的附件大小限制。上传成功后附件列表自动刷新。
5. 询价单附件仅包含采购员上传的文件，供应商报价附件归属于供应商报价单（详见需求5）。
6. 系统根据询价单物料的四级品类自动关联对应的询价单模版（来源于系统管理中的模版维护，按四级品类匹配）。采购员可在抬头中手动修改询价单模版，当询价单模版发生变化时，询价单附件中的询价模版文件自动更新为新选择的模版。

## 需求5：供应商报价单管理

### 报价单抬头字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 报价单号 | 文本（只读） | - | 系统自动生成 |
| 所属询价单号 | 文本（只读） | - | 关联的询价单编号 |
| 供应商编码 | 文本（只读） | - | 报价供应商编码 |
| 供应商名称 | 文本（只读） | - | 报价供应商名称 |
| 报价轮次 | 数字（只读） | - | 当前报价所属的询价轮次 |
| 报价状态 | 文本（只读） | - | 未报价/已报价/已过期 |
| 报价时间 | 日期时间（只读） | - | 供应商提交报价的时间 |
| 报价来源 | 文本（只读） | - | 邮件自动抓取/手工录入 |

### 报价明细行字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 物料编码 | 文本（只读） | - | 对应询价单物料明细行的物料编码 |
| 物料描述 | 文本（只读） | - | 对应询价单物料明细行的物料描述 |
| 数量 | 数字（只读） | - | 对应询价单物料明细行的数量 |
| 单位 | 文本（只读） | - | 对应询价单物料明细行的单位 |
| 报价含税单价 | 数字 | 必填 | 供应商对该物料的报价含税单价 |
| 报价含税金额 | 数字（只读） | - | 系统自动计算：报价含税单价 × 数量 |
| 币种 | 文本 | 必填 | 默认人民币，允许修改 |
| 税率 | 下拉选择框 | 必填 | 下拉选项：0.13、0.09、0.06、0.03、0.01、0，默认0.13 |
| 交货期 | 文本框 | 非必填 | 供应商承诺的交货期描述（如"30天"、"2026年5月底"） |
| 备注 | 文本 | 非必填 | 供应商报价备注信息 |

### 报价附件字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 附件名称 | 文本（展示） | - | 显示附件文件名 |
| 附件类型 | 文本（展示） | - | 显示文件类型（PDF/Excel/Word等） |
| 附件来源 | 文本（展示） | - | 邮件自动抓取/手工上传 |
| 上传时间 | 日期时间（展示） | - | 附件上传或抓取时间 |

### 逻辑说明

1. 每个供应商在每一轮询价中对应一张报价单，报价单与询价单通过询价单号关联。
2. 系统自动抓取供应商报价邮件后，自动创建报价单并填入报价数据。
3. 供应商报价邮件中的附件挂接到对应的报价单（非询价单），与该供应商的报价记录关联。
4. 采购员可手工录入报价数据时，同时可上传供应商提供的报价附件。
5. 报价附件支持查看、下载操作。
6. 报价含税金额由系统根据报价含税单价和数量自动计算。
7. 多轮询价时，每轮生成独立的报价单，历史轮次的报价单保留不覆盖。
8. 供应商报价详情区域提供供应商下拉选择框，切换供应商时标题、报价数据（单价/金额/交货期）、报价附件和报价时间同步更新。
9. 多轮报价按轮次降序展示（最新轮次在最上面），每轮报价区域包含轮次标题、报价时间、报价明细表和报价附件。

## 需求6：发送询价邮件

### 逻辑说明

1. 采购员点击【发送询价】按钮，系统通过SMTP协议实时向询价单中所有供应商的邮箱发送询价邮件。
2. 询价邮件使用标准化模板，包含：询价单编号、物料明细信息（物料编码、描述、数量、单位、需要日期）、报价截止时间。
3. 邮件附件包含采购员上传的相关附件文件。
4. 邮件发送使用采购员在"采购员信息维护"中配置的工作邮箱和授权码。
5. 发送成功后，询价单状态从"草稿"自动变更为"询价中"。
6. 若部分供应商邮件发送失败，系统提示失败的供应商列表，允许重新发送。
7. 邮件发送延迟 ≤ 5秒。

## 需求7：报价自动抓取与回写

### 逻辑说明

1. 系统通过IMAP/POP3协议定时抓取采购员邮箱中的供应商报价邮件。
2. 定时抓取任务的执行周期可配置。
3. 系统自动解析报价邮件内容，将报价数据回写到对应询价单的供应商明细中。
4. 系统自动抓取报价邮件中的附件，并将附件挂接到供应商对应的报价轮次中（即挂接到该供应商当前轮次的报价单）。
5. 首次收到报价数据回写后，询价单状态自动变更为"报价中"。
6. 所有供应商已报价、报价截止日期已过或采购员手动截止报价时，状态自动变更为"报价完成"。
7. 若报价邮件格式无法自动解析，支持AI辅助解析，解析结果需采购员人工确认。
8. 单次定时抓取任务执行时间 ≤ 5分钟。

---

## 需求8：多轮询价

### 逻辑说明

1. 采购员可对同一询价单进行多轮询价操作。
2. 每次发起新一轮询价时，系统自动递增当前轮次编号。
3. 新一轮询价时，采购员可调整报价截止时间（必须晚于当前时间）。
4. 系统保留每一轮的报价数据，支持跨轮次对比。
5. 采购员点击【发起新一轮询价】按钮，系统重新向供应商发送询价邮件。
6. 采购员可点击【截止报价】按钮提前终止当前轮次报价，系统将当前轮次所有未报价的供应商状态标记为"已过期"，询价单状态变更为"报价完成"。
7. 截止报价操作前弹窗确认："是否确认截止当前轮次报价？截止后未报价的供应商将无法继续报价。"

## 需求9：AI智能供应商推荐

### 逻辑说明

1. 在询价单详情页面，采购员点击【智能推荐供应商】按钮，触发AI智能供应商推荐智能体。
2. 智能体通过搜索引擎和天眼查等第三方平台API联网搜索与当前询价单物料相关的供应商。
3. 系统展示AI推荐的供应商列表，包含供应商名称、联系方式、经营范围等信息。
4. 推荐结果仅作为参考建议，采购员自行决定是否采纳。
5. 采购员可从推荐列表中选择供应商，点击【添加到询价单】一键将供应商添加到当前询价单的供应商明细行。
6. 添加的供应商来源标记为"AI推荐"。
7. AI推荐响应时间 ≤ 30秒。
8. 若第三方API不可用，系统提示"推荐服务暂时不可用"，不影响其他功能正常使用。

## 需求10：AI比价与策略推荐

### 报价对比表字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 物料编码 | 文本（展示） | - | 显示物料编码 |
| 物料描述 | 文本（展示） | - | 显示物料描述 |
| 物料详细描述 | 文本（展示） | - | 显示物料详细描述 |
| 数量 | 数字（展示） | - | 显示询价数量 |
| 需求日期 | 日期（展示） | - | 显示物料需求日期 |
| 单位 | 文本（展示） | - | 显示单位 |
| 最低价格 | 数字（展示） | - | 所有供应商最后一轮报价中的最低单价，自动计算并高亮 |
| 推荐中标供应商 | 下拉选择框 | - | 默认值为最低价格对应的供应商，采购员可手动修改 |
| 中标价格 | 文本框（可编辑） | - | 默认值为最低价格，采购员可手动修改。推荐中标供应商发生变更时，中标价格自动清空 |
| 供应商A-最终报价 | 数字（展示） | - | 供应商A最后一轮的报价含税单价 |
| 供应商A-第1轮报价 | 数字（展示） | - | 供应商A第1轮的报价含税单价 |

### 逻辑说明

1. 采购员点击【更新对比表】按钮，系统检索供应商当前轮次是否有报价信息，如果没有则自动抓取邮件更新报价数据，更新成功后提示"报价数据更新成功"。系统基于所有轮次的供应商报价数据自动生成报价对比表。
2. 对比表以矩阵形式展示：行为物料明细，列依次为物料信息（编码、描述、详细描述、数量、需求日期、单位）、汇总信息（最低价格、推荐中标供应商、中标价格）、各供应商报价（每个供应商先显示最终报价，再按轮次顺序显示各轮报价）。
3. 最低价格：取所有供应商最后一轮报价中的最低单价，自动计算并高亮显示。
4. 推荐中标供应商：系统自动推荐最后一轮报价中单价最低的供应商。
5. 中标价格：推荐中标供应商的最后一轮报价含税单价。
6. 最终报价：每个供应商的最后一轮报价含税单价，排在该供应商各轮次报价列的最前面。
7. 若某供应商在某轮次未报价，对应单元格显示"-"。
8. 支持按轮次筛选查看：可选择仅查看某一轮或全部轮次的对比数据。
9. 采购员点击【谈判策略建议】按钮，弹出谈判策略弹窗，展示AI生成的谈判策略（含整体分析、价格走势、谈判建议、推荐方案）。
10. 多轮询价结束后，采购员在报价对比表中通过推荐中标供应商下拉框和中标价格文本框确定中标结果。
11. 采购员可点击【导出对比表】按钮，将报价对比表导出为Excel文件。
12. 对比表和谈判策略生成时间 ≤ 15秒。
13. 谈判策略和中标推荐仅作为参考建议，采购员保留最终决策权。

## 需求11：询价结果导入ERP

### 逻辑说明

1. 采购员在"报价完成"状态下，导入ERP功能分为两个独立操作按钮：【报价单导入ERP】和【中标结果导入ERP】。
2. **报价单导入ERP：** 采购员点击【报价单导入ERP】按钮，系统将询价单中每个供应商的每一轮报价数据按轮次升序依次导入到Oracle EBS，导入数据中不体现轮次编号字段。此步骤中所有报价记录的"是否中标"字段均传"否"。同时将每轮报价单对应的附件一并推送到ERP系统。导入成功后询价单状态变更为"已定标"。
3. **中标结果导入ERP：** 报价单导入完成后（询价单状态为"已定标"），【中标结果导入ERP】按钮才可点击（报价单未导入时按钮置灰不可操作）。采购员点击【中标结果导入ERP】按钮，系统按供应商维度将中标供应商的报价信息导入到Oracle EBS，此步骤中中标供应商的"是否中标"字段传"是"。同时将中标供应商最后一轮报价单的附件推送到ERP系统。
4. 中标结果导入成功后，询价单状态自动变更为"已完成"。
5. 若任一步Oracle EBS API调用失败，系统自动重试（最多3次），重试仍失败则提示错误信息并记录异常日志。
6. 关键操作记录审计日志。

### 报价单推送ERP接口规范

| 项目 | 说明 |
|------|------|
| 调用方向 | 询比价系统 → Oracle EBS |
| 调用方式 | 按供应商维度，每个供应商每一轮报价调用一次，按轮次升序依次调用 |

**传参抬头：**

| 参数 | 类型 | 说明 |
|------|------|------|
| 供应商编码 | 文本 | 询价单中的供应商编码 |
| 是否年度合同 | 是/否 | 取询价单抬头中的"是否年度谈判"字段值 |
| 是否末次报价 | 是/否 | 如果是该供应商最后一轮报价则为"是"，其余轮次为"否" |
| 是否中标 | 是/否 | 报价单导入时全部传"否"；中标结果导入时，中标供应商传"是" |
| 附件 | 文件 | 该供应商当前轮次报价单对应的附件文件 |

**传参明细（按物料行传入）：**

| 参数 | 类型 | 说明 |
|------|------|------|
| 物料编码 | 文本 | 报价单明细中的物料编码 |
| 含税单价 | 数字 | 报价单明细中当前轮次的报价单价 |
| 税率 | 数字 | 报价单明细中的税率 |
| 数量 | 数字 | 报价单明细中的数量 |
| 交货期 | 文本 | 报价单明细中的交货期 |

### 逻辑说明（续）

4. 报价单导入和中标结果导入均使用上述接口规范，区别在于"是否中标"字段的取值不同。

### ERP接收报价单后续处理逻辑

1. ERP系统接收到报价单数据后，判断"是否中标"字段。
2. 如果报价单为中标报价单（是否中标="是"），ERP系统自动将对应的报价单发起OA采购报价审批流程。
3. 非中标报价单（是否中标="否"）仅做数据存档，不触发OA审批流程。
5. 若任一步Oracle EBS API调用失败，系统自动重试（最多3次），重试仍失败则提示错误信息并记录异常日志，询价单状态保持不变。
6. 关键操作记录审计日志。

## 需求12：询价单状态流转

### 状态说明

| 状态 | 说明 | 允许的操作 |
|------|------|-----------|
| 草稿 | 询价单已创建，可自由编辑 | 编辑全部信息、添加供应商、上传附件、发送询价、删除、作废 |
| 询价中 | 询价邮件已发送，等待供应商报价 | 查看、上传附件 |
| 报价中 | 已收到部分供应商报价 | 查看报价、手工录入报价、截止报价 |
| 报价完成 | 所有供应商已报价、报价截止日期已过或手动截止 | 更新对比表、谈判策略建议、导出对比表、报价单导入ERP、发起新一轮询价 |
| 已定标 | 报价单已导入ERP，等待中标结果导入 | 中标结果导入ERP、谈判策略建议、导出对比表、查看 |
| 已完成 | 询价结果已导入ERP | 只读查看 |
| 已作废 | 询价单已作废 | 只读查看 |

### 状态流转规则

```
草稿 → 询价中（采购员点击【发送询价】，邮件发送成功）
询价中 → 报价中（系统抓取到第一个供应商报价）
报价中 → 报价完成（所有供应商已报价 或 报价截止日期已过 或 采购员点击【截止报价】）
报价完成 → 已定标（采购员点击【报价单导入ERP】，报价单数据导入成功）
已定标 → 已完成（采购员完成【中标结果导入ERP】）
报价完成 → 询价中（采购员点击【发起新一轮询价】，邮件发送成功）
草稿 → 已作废（采购员点击【作废】）
询价中/报价中/报价完成/已定标 → 已作废（采购员在详情页点击【作废】）
```

### 逻辑说明

1. 询价单创建后初始状态为"草稿"。
2. 采购员在草稿状态下点击【发送询价】，邮件发送成功后状态变更为"询价中"。
3. 系统抓取到第一个供应商报价后状态变更为"报价中"。
4. 所有供应商已报价、报价截止日期已过或采购员手动截止报价，状态变更为"报价完成"。
5. 采购员点击【报价单导入ERP】将报价数据导入ERP后状态变更为"已定标"。
6. 采购员点击【中标结果导入ERP】将中标结果导入ERP后状态变更为"已完成"。
8. 发起新一轮询价时，状态回退为"询价中"，轮次编号递增。
9. 所有未完成状态的询价单均可在详情页作废，草稿状态还可在首页批量作废。
10. 状态流转由系统自动控制，采购员不可手动修改状态。

## 需求13：询价单删除与作废

### 逻辑说明

**删除：**
1. 仅"草稿"状态的询价单允许删除。
2. 点击【删除】按钮，弹窗提示"是否确认删除该询价单？删除后不可恢复。"
3. 点击【确定】，删除询价单及其所有物料明细、供应商明细和附件。
4. 点击【取消】，关闭弹窗，取消操作。

**作废：**
5. 所有未完成状态的询价单均允许作废（草稿/询价中/报价中/报价完成/已定标）。
6. 询价中及之后状态的询价单只能在询价单详情页面中执行作废操作。
7. 点击【作废】按钮，弹窗提示"是否确认作废该询价单？作废后不可恢复。"
8. 点击【确定】，询价单状态变更为"已作废"，数据保留但不可再操作。
9. 已完成和已作废状态的询价单不允许再次作废。

## 需求15：报价数据手工录入

### 逻辑说明

1. 当供应商通过非邮件方式（如电话、传真）提供报价时，采购员可手工录入报价数据。
2. 仅在"询价中"或"报价中"状态下允许手工录入报价，其他状态下【手工录入报价】按钮置灰不可操作。
2. 点击供应商明细行的【手工录入报价】按钮，弹出报价录入表单。
3. 报价录入表单自动带入当前询价轮次，手工录入的报价数据归属到当前轮次的报价单中。
4. 采购员逐条录入每个物料的报价含税单价。
5. 手工录入报价时，支持采购员上传供应商报价附件，附件挂接到该供应商当前轮次的报价单中。
6. 手工录入的报价标记来源为"手工录入"，与系统自动抓取的报价区分。
7. 手工录入报价后，该供应商在当前轮次的报价状态变更为"已报价"。

## 需求16：邮件发送记录

### 逻辑说明
