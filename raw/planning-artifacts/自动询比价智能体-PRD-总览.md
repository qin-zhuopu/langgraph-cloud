## 概述

自动询比价智能体是一个AI驱动的采购智能平台，旨在将采购人员从繁琐的手工询比价操作中解放出来。系统实现从询价到定标的全生命周期自动化，包含智能供应商推荐、自动创建询价单、邮件询价、报价自动抓取、AI比价分析与谈判策略推荐，以及ERP结果回写。

---

## 系统角色

| 角色 | 权限范围 | 数据访问 |
|------|---------|---------|
| IT管理员 | 系统管理、账号创建、权限分配 | 全局系统配置 |
| 采购员 | 询价全流程操作 | 仅自己的数据 |

---

## 系统架构概要

- **项目类型：** SaaS/B2B Web应用
- **多租户：** 共享数据库，基于Oracle EBS业务实体进行租户隔离
- **部署方式：** 私有化部署
- **ERP对接：** Oracle EBS（API方式）
- **邮件系统：** SMTP发送 / IMAP定时抓取
- **AI数据源：** 搜索引擎、天眼查等第三方平台（API方式）

## 非功能需求

### 性能

| 指标 | 要求 |
|------|------|
| 页面加载和用户操作响应时间 | ≤ 3秒 |
| 询价邮件发送延迟 | ≤ 5秒 |
| 报价邮件定时抓取单次执行时间 | ≤ 5分钟 |
| 并发采购员支持 | 60+ |
| AI智能供应商推荐响应时间 | ≤ 30秒 |
| 报价对比表和谈判策略生成时间 | ≤ 15秒 |

### 安全

- 邮箱授权码加密存储，不得明文保存或传输
- 所有数据传输使用HTTPS加密
- 用户认证采用安全的身份验证机制（如JWT Token）
- 采购员数据严格隔离，不可跨用户访问
- 租户数据严格隔离，不可跨租户访问
- 关键操作（发送询价、修改报价、定标、ERP回写）记录审计日志

### 可用性与可靠性

| 指标 | 要求 |
|------|------|
| 系统可用性 | ≥ 99.5% |
| 邮件发送成功率 | ≥ 99% |
| 报价抓取定时任务执行成功率 | ≥ 95% |
| Oracle EBS API调用失败重试 | 最多3次 |

### 集成

- Oracle EBS API集成支持超时处理和重试机制
- 天眼查等第三方API不可用时，系统其他功能不受影响（降级处理）
- 邮件系统支持SMTP发送和IMAP/POP3接收协议
- 私有化部署环境需确保外部API的网络可达性

## 风险与缓解策略

| 风险类型 | 风险描述 | 缓解措施 |
|---------|---------|---------|
| 技术风险 | AI智能体推荐质量不稳定 | AI推荐仅作参考，采购员保留最终决策权 |
| 技术风险 | 报价邮件格式多样解析困难 | AI辅助解析+人工确认兜底 |
| 集成风险 | Oracle EBS API接口不稳定 | 自动重试+异常告警+降级方案 |
| 集成风险 | 天眼查等第三方API限制或变更 | 多数据源支持，降低单一依赖 |
| 安全风险 | 邮箱授权码泄露 | 加密存储+定期更换提醒 |
| 安全风险 | 采购数据越权访问 | 数据库+应用层双重数据隔离 |
