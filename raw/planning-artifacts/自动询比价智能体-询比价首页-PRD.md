# 自动询比价智能体 — 询比价首页需求文档

**作者：** 15481
**日期：** 2026-04-15
**模块：** 自动询比价智能体 > 询比价首页

---

## 概述

询比价首页是采购员登录系统后的主界面，展示采购员正在进行的询价信息，并提供自动创建询价单和手工创建询价单的入口，同时提供AI智能体（智能供应商推荐、自动比价及采购策略推荐）的访问入口。

---

## 需求1：进行中询价单列表展示

### 字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 询价单状态 | 文本（展示） | - | 显示当前状态，详见状态说明 |
| 报价单号 | 文本（展示） | - | 系统自动生成的唯一编号 |
| 业务实体 | 文本（展示） | - | 显示询价单所属的Oracle EBS业务实体 |
| 报价单标题 | 文本（展示） | - | 显示询价单标题（由采购员填写或系统自动生成） |
| 采购员 | 文本（展示） | - | 显示询价单所属采购员姓名 |
| 报价截止时间 | 日期时间（展示） | - | 格式为yyyy-mm-dd HH:mm:ss |
| 备注 | 文本（展示） | - | 显示询价单备注信息 |
| 创建人 | 文本（展示） | - | 显示创建该询价单的操作人 |
| 创建日期 | 日期（展示） | - | 格式为yyyy-mm-dd |
| 是否年度谈判 | 文本（展示） | - | 显示是/否 |

### 询价单状态说明

| 状态 | 说明 |
|------|------|
| 草稿 | 询价单已创建，可自由编辑所有信息 |
| 询价中 | 询价邮件已发送，等待供应商报价 |
| 报价中 | 已收到部分供应商报价，仍有供应商未报价 |
| 报价完成 | 所有供应商已报价或报价截止日期已过 |
| 已定标 | 报价单已导入ERP，等待中标结果导入 |
| 已完成 | 询价结果已导入ERP，流程结束 |
| 已作废 | 询价单已作废，不再继续 |

### 逻辑说明

1. 以列表形式展示当前采购员的询价单，仅显示当前登录采购员自己的数据。
2. 列表支持分页展示。
3. 点击某条询价单可进入询价单详情页面。
4. 列表按创建日期倒序排列，最新创建的询价单排在最前面。
5. 询价单状态以不同颜色标签区分显示。

## 需求2：自动创建询价单

### 逻辑说明

1. 点击【自动创建询价单】按钮，系统通过API从Oracle EBS抓取当前采购员待采购的物料数据。

   **接口规范：**

   | 项目 | 说明 |
   |------|------|
   | 调用方向 | 询比价系统 → Oracle EBS |
   | 传入参数 | 采购员工号 |
   | 返回参数 | 业务实体、物料编码、物料描述、物料详细说明、需求数量、单位、需要日期、四级品类 |

   **ERP抓取逻辑：**
   - 取当前采购员所有业务实体下正常料号的请购池信息。
   - 按业务实体、物料编码、需求日期进行汇总排重。
   - 需求数量为汇总值（同一业务实体+物料编码+需求日期的请购数量求和）。

2. 系统根据物料的业务实体和四级品类自动进行分组，同一业务实体且同一四级品类的物料归入同一询价单。
3. 系统根据四级品类自动匹配采购员已维护的供应商组，将对应供应商组中的供应商明细自动填入询价单的供应商明细行。
4. 分组和供应商匹配为一步操作，完成后自动生成带有物料明细和供应商明细的询价单。
5. 系统自动生成询价单号，编码规则为：6位日期（YYMMDD）+ 4位流水码，流水码每天从0001开始重新编排。例如：2604150001、2604150002。
6. 生成的询价单抬头自动填入业务实体信息（来源于Oracle EBS）。
6. 采购员可查看自动生成的询价单，确认或调整后进入询价单详情进行后续操作。
7. 自动创建完成后，新生成的询价单（状态为"草稿"）自动显示在首页询价单列表中，列表自动刷新。
8. 若Oracle EBS API调用失败，系统提示错误信息并支持重试。

## 需求3：手工创建询价单

### 物料录入字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 业务实体 | 下拉选择框 | 必填 | 调用采购员有权限的业务实体列表，下拉选择 |
| 物料编码 | 文本框 | 必填 | 手动输入物料编码，输入后结合业务实体调用接口自动获取物料信息 |
| 物料描述 | 文本（只读） | - | 由接口自动返回，不可编辑 |
| 物料详细描述 | 文本（只读） | - | 由接口自动返回，不可编辑 |
| 数量 | 数字框 | 必填 | 手动输入，必须大于0 |
| 单位 | 文本（只读） | - | 由接口自动返回，不可编辑 |
| 需要日期 | 日期选择器 | 必填 | 格式为yyyy-mm-dd |
| 四级品类 | 文本（只读） | - | 由接口自动返回，不可编辑 |

### 物料信息查询接口规范

| 项目 | 说明 |
|------|------|
| 调用方向 | 询比价系统 → Oracle EBS |
| 传入参数 | 业务实体、物料编码 |
| 返回参数 | 物料描述、物料详细描述、单位、四级品类 |

**ERP抓取逻辑：**
- 根据业务实体和物料编码查询已生效的物料对应的物料描述、物料详细描述、单位、四级品类。
- 若物料编码在该业务实体下不存在或未生效，系统提示错误信息。

### 逻辑说明

1. 点击【手工创建询价单】按钮，进入物料录入界面。
2. 采购员先选择业务实体（下拉选择框，仅显示采购员有权限的业务实体）。
3. 采购员输入物料编码后，系统自动调用Oracle EBS接口，传入业务实体和物料编码，获取物料描述、物料详细描述、单位、四级品类，这些字段自动填入且不可编辑。
4. 采购员手动填写数量和需要日期。
5. 支持录入多条物料信息。
6. 必填字段未填写时，阻止提交并提示用户填写。

### Excel导入功能

1. 点击【Excel导入】按钮，下载导入模板。导入模板包含两列：业务实体、物料编码。
2. 采购员填写模板后上传Excel文件。
3. 系统逐行读取Excel数据，调用Oracle EBS接口（传入业务实体和物料编码），自动获取每行物料的物料描述、物料详细描述、单位、四级品类信息。
4. 导入成功的物料自动填入物料录入列表，采购员需补充数量和需要日期。
5. 导入失败的行（如物料编码不存在或未生效）标记错误原因，展示导入结果：成功条数、失败条数及失败原因明细。

### 创建询价单

1. 物料信息录入完成后，采购员点击【创建询价单】按钮。
2. 系统根据业务实体和四级品类自动进行分组，同时匹配供应商组中的供应商，生成询价单。
3. 分组和供应商匹配逻辑与自动创建询价单一致。
4. 询价单号自动生成，编码规则与自动创建一致：6位日期（YYMMDD）+ 4位流水码。
5. 创建完成后，新生成的询价单（状态为"草稿"）自动显示在首页询价单列表中，列表自动刷新。

## 需求4：AI智能体 - 智能供应商推荐

### 逻辑说明

1. 在询价单列表上方，展示【智能供应商推荐】AI智能体入口卡片。
2. 点击【智能供应商推荐】入口，打开智能供应商推荐智能体对话界面。
3. 采购员可通过对话方式描述采购需求，智能体根据物料四级品类进行供应商推荐。
4. 供应商推荐分为两部分：
   - **ERP内部推荐：** 系统首先校验当前采购员在本业务实体下是否采购过该四级品类的物料。若采购过，则从Oracle EBS中查询其他业务实体下已有该四级品类供货记录、但尚未在当前业务实体下下单的供应商进行推荐；若未采购过该四级品类，则不推荐其他业务实体的供应商。
   - **外部联网推荐：** 通过搜索引擎和天眼查等第三方平台API联网搜索与该四级品类匹配的供应商。
5. 推荐展示的供应商个数根据采购员在"采购员信息维护"中配置的推荐个数进行控制。
6. 推荐结果仅作为参考建议，采购员自行决定是否采纳。
7. 智能体的详细功能逻辑详见询价单详情模块文档中的需求9。

## 需求5：AI智能体 - 自动比价与策略推荐

### 逻辑说明

1. 在询价单列表上方，展示【自动比价与策略推荐】AI智能体入口卡片。
2. 点击【自动比价与策略推荐】入口，打开自动比价智能体对话界面。
3. 采购员可选择询价单，智能体自动分析供应商报价数据并生成比价对比表和谈判策略建议。
4. 谈判策略和中标推荐仅作为参考建议，采购员保留最终决策权。
5. 智能体的详细功能逻辑详见询价单详情模块文档中的需求10。

## 需求6：询价单搜索与筛选

### 筛选条件字段说明

| 字段释义 | 字段类型 | 是否必填 | 字段规则说明 |
|---------|---------|---------|-------------|
| 询价单编号 | 文本框 | 非必填 | 支持模糊搜索 |
| 询价单状态 | 下拉单选 | 非必填 | 下拉选项：全部/草稿/询价中/报价中/报价完成/已定标/已完成/已作废 |
| 创建时间范围 | 日期范围选择器 | 非必填 | 选择起止日期 |
| 业务实体 | 下拉单选 | 非必填 | 下拉选项：当前采购员关联的业务实体 |

### 逻辑说明

1. 列表上方提供搜索和筛选区域。
2. 支持按询价单编号模糊搜索。
3. 支持按询价单状态、创建时间范围、业务实体进行筛选。
4. 筛选条件支持组合使用。
5. 点击【重置】按钮清空所有筛选条件，恢复默认列表。

## 需求7：Tab页签切换

### 逻辑说明

1. 首页提供【全部】、【草稿】、【进行中】、【已完成】四个Tab页签切换。
2. 【全部】页签展示所有状态的询价单（默认显示）。
3. 【草稿】页签展示状态为草稿的询价单。
4. 【进行中】页签展示状态为询价中/报价中/报价完成/已定标的询价单。
5. 【已完成】页签展示状态为已完成和已作废的询价单。
6. 四个页签均支持搜索、筛选和分页功能。
7. 已完成和已作废的询价单点击后进入只读详情页面。

---

## 需求8：ERP数据预览与确认

### 逻辑说明

1. 点击【自动创建询价单】后，系统先展示从ERP抓取的物料数据预览列表，不直接生成询价单。
2. 采购员可在预览列表中勾选需要询价的物料，取消不需要的物料。
3. 采购员确认物料选择后，点击【确认生成】按钮，系统再执行四级品类分组和供应商匹配。
4. 若ERP中无待采购物料数据，系统提示"当前无待采购物料"。

---

## 需求9：询价单批量操作

### 逻辑说明

1. 列表支持多选（勾选框）。
2. 支持批量删除：勾选多条草稿状态的询价单后，点击【批量删除】按钮，弹窗确认后批量删除。
3. 仅草稿状态的询价单允许删除操作。
4. 支持批量作废：勾选多条草稿状态的询价单后，点击【批量作废】按钮，弹窗确认后批量作废。
5. 询价中及之后状态的询价单不允许在首页批量操作，只能在询价单详情中进行作废。
